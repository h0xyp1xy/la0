# Левушкин — сайт продажи картины

## Быстрый старт

- Откройте `index.html` в браузере.

## Сохранение заявок в базу (Django)

Чтобы данные формы заказа попадали в базу данных:

1. Установите зависимости: `pip install -r requirements.txt`
2. Перейдите в каталог бэкенда: `cd backend`
3. Примените миграции (если ещё не применены): `python manage.py migrate`
4. Запустите сервер: `python manage.py runserver`
5. Откройте в браузере: http://127.0.0.1:8000/

Заявки сохраняются в SQLite (`backend/db.sqlite3`). Просмотр в админке: создайте суперпользователя `python manage.py createsuperuser` и откройте http://127.0.0.1:8000/admin/

## Что менять

- **Фото картины**: замените файл `assets/painting.svg` на ваше изображение (можно `.jpg/.png/.webp`), и обновите `src` у тега `<img>` в `index.html` при необходимости.
- **Контакты**: обновите телефон и Telegram в `index.html` (в верхней и нижней панели).
- **Текст и цена**: обновите блок в центре страницы (название/описание/цена) в `index.html`.
